CMPS 161 Fall 2013 Final Project

Project 1 Extension: Implementing streaklines to show wind space traveled

 

 

 

Source Code: finalprogsource.html

Technical Document: Final Project Technical Document.pdf

New data set ( wind taken over multiple hours from multiple stations):stations.json

 

 

Code Structure:

D3 was used to create visualization.

Program is arranged by “mode” of display, with code for implementing streaklines going from line 360 to 454.

Improvements from project 1: Implemented display of data from multiple hours of time. Implemented display of streaklines.

User Guide:

Use bar on right side of screen to determine time period for which data is displayed. Use mode button to change display from stations, to velocity glyphs, to streamlines, to pathlines.
